# Outputs Layout

This `outputs` directory is now split into two top-level buckets:

- `paper_final/`
  - curated outputs kept for paper writing and reporting
- `archive_exploration/`
  - older smoke runs, probes, regression experiments, and exploratory outputs

If you only need the final paper tables, start from:

- `paper_final/README.md`

If you need historical debugging or optimization traces, look in:

- `archive_exploration/`
